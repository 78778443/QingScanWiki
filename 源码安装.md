## 1. 下载QingScan
```

https://github.com/78778443/QingScan  
```
进入目录qing-scan/code
```
cd qing-scan/code
```



## 2. 安装扩展
```
apt install php-intl php-pdo php-mysqli
```
## 3. 安装依赖
```
composer install
```
### 4. 导入数据




## 4. 运行项目
```
cd /qing-scan/code && php think run
```

接下来通过浏览器去访问URL地址：`http://host:8000/`

点击生成token

![](images/20211206164002.png)

复制GitHub中生成的token

![](images/20211206164143.png)
